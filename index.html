This project was created using p5.js and is hosted by GitHub Pages. All code (aside from the p5 libraries) was written from scratch. All sprites were (unfortunately) created manually (give me a break, I'm no graphic designer).

Click a or m to toggle between AI mode and manual.

Manual mode:
Click space or left-mouse to jump, r to restart, p to pause.
